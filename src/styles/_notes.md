# src/styles/

## global.css
Tailwind CSS entry point. Also includes base resets and any global element styles.

## tokens.css
Brand design tokens as CSS custom properties. Import this in global.css.
Defines colors, font families, and spacing scale.

## Brand Color Palette
| Token | Hex | Use |
|---|---|---|
| --color-gold | #C9A84C | Primary accent, CTAs, highlights |
| --color-gold-light | #E8C97A | Hover states, backgrounds |
| --color-black | #1A1A1A | Primary background |
| --color-charcoal | #2D2D2D | Secondary background, cards |
| --color-red | #8B1A1A | Accent, section headers |
| --color-cream | #FAF6EE | Light background sections |
| --color-white | #FAFAFA | Text on dark backgrounds |

## Typography
- Display/headings: Georgia or a premium serif (e.g., Playfair Display via Google Fonts)
- Body: System font stack or Inter — clean and readable at small sizes
- The logo uses a decorative script — rendered as an image, not a font
